## 🌟 Overview

**Video Downloader Pro Max** is a high-performance browser extension (Manifest V3) engineered by **Yehia (Sa3dny Team)**. It is designed to provide a seamless, premium video downloading experience across virtually every major video platform. Unlike standard downloaders, the **Pro Max** edition comes with **Premium Features Unlocked** and advanced media detection algorithms.

## 🛠️ Technical Architecture

This project is built using the latest browser extension standards and high-performance libraries:

- **Manifest V3:** Ensuring future-proof compatibility and security.
- **WebAssembly (WASM):** Integration of `libav` for complex media operations directly in the browser.
- **Service Workers:** Efficient background processing for handling multiple downloads.
- **Shadow DOM:** Non-intrusive UI overlays for a clean user experience.

## 📥 Installation Guide

### For Developers / Manual Installation
Since this is the **Pro Max** source/unpacked version, follow these steps:

1. **Download the Repository:** Clone or download the ZIP file and extract it to a folder.
2. **Open Extensions Page:** Navigate to `chrome://extensions/` in your browser.
3. **Enable Developer Mode:** Toggle the **Developer mode** switch in the top right corner.
4. **Load Unpacked:** Click the **"Load unpacked"** button.
5. **Select Folder:** Choose the folder where you extracted the project files.
6. **Done!** The extension icon will appear in your toolbar.

## 🚀 How to Use

1. **Navigate:** Visit any website containing a video.
2. **Detect:** The extension icon will light up or show a badge when media is detected.
3. **Choose Quality:** Click the icon to open the **Pro Max Panel**. Select your desired resolution or format.
4. **Download:** Click the download button. You can monitor progress in the built-in **History** or **Side Panel**.
5. **Smart Naming:** Use the integrated smart naming tool to automatically organize your files.
